API do Instagram

Conheça a API do Instagram
Comece a usar a API do Instagram para criar, publicar e gerenciar conteúdo. Para interagir com os usuários, você também pode enviar e receber mensagens, além de moderar comentários. Saiba mais
Se quiser rastrear hashtags e insights, mude para o API setup with Facebook login.

1. Adicionar permissões obrigatórias de mensagens
Adicione as permissões listadas para criar, publicar e gerenciar conteúdo, além de responder diretamente a mensagens e enviar respostas privadas com sua conta do Instagram. As permissões e os recursos obrigatórios e opcionais podem ser configurados na página Permissões e recursos.
instagram_business_basic
instagram_manage_comments
instagram_business_manage_messages
2. Gerar tokens de acesso
Adicione uma conta do Instagram para gerar tokens de acesso e configurar assinaturas de webhook. Antes de prosseguir, atribua a função de Testador do Instagram à conta na aba Funções.
3. Configurar webhooks
Configure um URL de webhook personalizado ou use serviços para configurar um ponto de extremidade. Saiba mais
Seu app deve estar com o status de publicado para receber webhooks.
URL de callback
https://www.politicall.com.br/api/webhook/instagram

4. Configurar o login da empresa no Instagram
Forneça uma forma segura para as empresas concederem permissões ao seu app para acessar dados com o login da empresa.
5. Concluir a análise do app
Para que seu app acesse dados ao vivo, o Instagram exige que o processo de análise do app seja realizado. Passe por esse processo para ter acesso avançado às permissões do Instagram. Envie sua solicitação de análise do app quando tudo estiver pronto.
